## Message containers

```md
::: tip
This is a tip
:::

::: warning
This is a warning
:::

::: danger
This is a danger warning
:::
```

#### Example

::: tip
This is a tip
:::

::: warning
This is a warning
:::

::: danger
This is a danger warning
:::

You can also customize the title of the block:

```md
::: danger STOP
Danger, do not proceed
:::
```

#### Example

::: danger STOP
Danger, do not proceed
:::

````md

};

````
